# PyChat

A simple chat application built in Python during my sophomore year programming class. This project is fully open source—feel free to use, modify, and distribute it.

## Features
- Basic chat functionality (details depend on implementation).
- Lightweight and easy to run.

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/SkylarMyers/PyChat.git
    ```
2. Navigate to the directory:
    ```
    cd PyChat
    ```
3. Install dependencies (if any):
    ```
    pip install -r requirements.txt
    ```

## Usage
Run the main script:
```
python main.py
```
Follow on-screen instructions to start chatting.
